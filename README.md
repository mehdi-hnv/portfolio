# Mehdi Portfolio Website

Modern, responsive multi-page site showcasing Mehdi's background, projects, and resume snapshot.

## Pages
- `index.html` — home landing with hero, highlights, and social links
- `about.html` — background, qualifications, and timeline
- `projects.html` — detailed cards for Project 1, Hour of Code, and more
- `resume.html` — skills, experience, and collaboration call-to-action

## Structure
```
assets/
  css/
    reset.css    # base reset styles
    styles.css   # theme, layout, responsive rules
  images/        # SVG profile + project art
  js/
    main.js      # navigation toggle, footer year helper
```

## Getting Started
1. Serve locally with any static server, e.g. `npx serve .` or VSCode Live Server.
2. Update content (photos, copy, project links) to match your latest info.
3. Update the GitHub repository links inside each page if your remote URL differs.

## Deployment
Host on GitHub Pages, Netlify, or Vercel. Ensure the GitHub repo that powers this site is public and linked within the footer.
